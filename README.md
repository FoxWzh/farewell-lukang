## 安装

**Claude Code：**
```bash
claude plugin install github:FoxWzh/farewell-lukang
```

**Codex / 其他 agent：**
```bash
git clone https://github.com/FoxWzh/farewell-lukang
```
把仓库根目录加入 agent 的上下文，`AGENTS.md` 会被自动读取。

## 使用

```
召唤陆抗
```

或者 `我要找陆抗`，Claude / Codex 切换成陆抗身份。
